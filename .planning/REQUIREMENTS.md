# Requirements

## 1. 1차 MVP (우선순위: 매우 높음)

- [ ] **촬영 가이드 제공**: 복잡한 설명 대신 5-10초의 촬영 방법, 60fps 권장, 공과 사용자가 잘 나오는 구도 안내
- [ ] **영상 업로드/촬영 기능**: 5~10초 분량의 영상 파일(mp4, mov) 업로드 지원
- [ ] **온디바이스 포즈 분석**: MediaPipe 등 활용하여 주요 관절 좌표(키포인트) 추출
- [ ] **공 검출 및 공속도 추정**:
  - 임팩트 순간 탐지
  - 사용자가 입력한 프리킥 거리 기반 평균 공속도(km/h) 산출
  - 신뢰도(높음/보통/낮음) 등급 산출
- [ ] **분석 엔진/스코어링**:
  - 전체 프리킥 점수 산출
  - 킥 타입 분류 (예: 인스텝 파워형)
  - 개선 포인트 3가지 제공 (자세 불안정, 팔로스루 상태 등)
- [ ] **결과 공유 기능**: 기기 로컬에서 공유용 요약 결과 카드(스틸 이미지, 점수, 공속도 포함) 렌더링
- [ ] **개인정보 보호**: 기본적으로 원본 영상 미저장 정책(동의 없는 서버 전송 불가)

## 2. 2차 MVP (우선순위: 중간)

- [ ] 사용자 기록 저장 기능 (이전 영상과 분석 결과 비교)
- [ ] 거리별/사용자별 랭킹 대시보드
- [ ] 사용자 동의 기반 비식별화 데이터수집 동의 플로우
- [ ] 공 검출 실패 시 사용자가 직접 2~3개 프레임에서 공을 터치해 위치를 지정하는 보정 UI 추가

## 3. 3차 MVP (우선순위: 낮음)

- [ ] 팀/코치용 관리 기능 (여러 선수 데이터 묶음 관리)
- [ ] 유료 리포트(상세 지표, PDF 다운로드) 제공
- [ ] 120fps 고정밀 분석 모드 지원
- [ ] 고급 3D 공 궤적 분석 시스템
