# Phase 4: 결과 화면 및 공유 기능

## 목적 (Objective)
사용자가 자신의 프리킥 능력을 다른 사람에게 직관적이고 멋지게 공유할 수 있도록, 분석 결과를 요약한 '스탯 카드(Stat Card)' 형태의 공유 전용 UI를 구성하고 이를 이미지로 캡처(Web Share API 연동)하는 기능을 제공합니다.

## 작업 목록 (Tasks)

1. **외부 패키지 설치**
   - `html2canvas` 라이브러리를 설치하여 DOM 엘리먼트를 이미지(Blob)로 변환하는 캡처 기능을 준비합니다.

2. **공유용 스탯 카드 컴포넌트 설계 (`components/ShareCard.tsx`)**
   - 분석 결과 데이터(구속, 폼 스코어, 킥 타입 등)를 Props로 받아 렌더링하는 숨김 처리용(또는 오프스크린) React 컴포넌트를 설계합니다.
   - 네온 컬러 디자인, 로고, "Freekick Master" 브랜딩 텍스트가 포함된 다이내믹한 레이아웃을 구성합니다.

3. **캡처 및 공유 로직 구현 (`app/analyze/page.tsx` 연동)**
   - 기존의 "Share" 버튼을 눌렀을 때, `html2canvas`를 통해 `ShareCard` 영역을 캡처하여 `.png` 형태의 File 객체로 변환합니다.
   - 생성된 파일을 `navigator.share({ files: [file] })` 호출을 통해 인스타그램, 카카오톡 등에 넘기거나, 해당 브라우저가 지원하지 않을 경우 `<a>` 태그의 `download` 속성을 사용해 이미지 파일로 자동 저장(Fallback)되도록 처리합니다.

## 검증 방법 (Verification)
- 모바일(또는 데스크톱) 브라우저에서 'Share' 버튼 클릭 시, 화면에 잠시 로딩 스피너가 표시되고 곧바로 네이티브 공유 팝업(또는 다운로드 팝업)이 열리는지 확인.
- 공유/저장된 이미지를 열었을 때 글씨가 깨지지 않고 네온 디자인이 잘 보존되어 있는지 캡처 품질 검증.
