# Free-Kick Analysis App

## 핵심 결론 (Core Vision)
"내 프리킥 영상만 올리면, 공속도와 자세 문제를 30초 안에 분석해주는 앱"

본 프로젝트는 정밀 계측보다는 **조건부 추정 및 실질적 피드백 제공**에 초점을 맞춥니다.
사용자가 자신의 슛을 찍고, 개선점을 시각적으로 확인하며, 그 결과를 친구들과 공유할 수 있도록 돕는 것이 핵심 가치입니다.

## 아키텍처 원칙 (Architecture Constraints)
- **온디바이스 중심**: 원본 영상은 기기에서 직접 처리(TensorFlow.js, MediaPipe, WebAssembly 등 활용)하며 기본적으로 서버에 업로드/저장하지 않습니다.
- **저비용 & 확장성**: PWA/Next.js를 기반으로 구성하고, 백엔드는 메타데이터(키포인트, 공 좌표, 익명 세션 정보 등)만 저장하여 인프라 비용(GPU, 트래픽, 스토리지)을 최소화합니다.
- **사용자 동의 기반 데이터 수집**: 자체 모델 개선을 위해 영상 저장이 필요할 경우, 명시적으로 동의한 사용자에 한해서만 비식별화된 데이터로 처리합니다.

## 분석 엔진 구조
1. 영상 메타데이터 추출 (FPS, 해상도 확인)
2. 사람 포즈 추출 (MediaPipe / YOLO Pose)
3. 공 검출 (YOLO Ball Detector)
4. 임팩트 순간 자동 탐지
5. 필드/골대 보정 (픽셀-미터 변환)
6. 공속도 추정 (비행 시간 기반 평균 속도 + 초기 이동량 기반 순간 속도)
7. 프리킥 점수/자세 평가 리포트 생성 및 공유 이미지 제작

## 데이터 활용 전략
- 오픈 데이터(SoccerKicks, Wyscout, IDSSE 등)는 초기 분석 엔진의 벤치마크 및 기준 스탯 구축에만 제한적으로 사용합니다.
- 상용 서비스용 모델은 향후 사용자 동의하에 수집된 데이터를 통해 점진적으로 고도화합니다.
