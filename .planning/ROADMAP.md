# Roadmap

## Phase 1: 기반 설정 및 프론트엔드 환경 구성 (1차 MVP)
- [ ] Next.js 기반 PWA 스캐폴딩 설정
- [ ] 테일윈드(또는 지정된 CSS 솔루션)와 UI 컴포넌트 시스템 구축
- [ ] 영상 업로드 및 로컬 렌더링을 위한 파일 시스템 처리 로직 구현
- [ ] 기본 라우팅 및 촬영 가이드 화면 개발

## Phase 2: 온디바이스 AI 파이프라인 (1차 MVP)
- [ ] MediaPipe(혹은 대체제) 포즈 추정 모델 Web Worker 연동
- [ ] 영상 프레임 추출 및 캔버스 오버레이 드로잉
- [ ] 경량화된 공 검출(Ball Detection) 모델 포팅 및 연동
- [ ] 사용자 입력을 바탕으로 임팩트 타이밍 휴리스틱/로직 구현

## Phase 3: 분석 엔진 및 스코어링 (1차 MVP)
- [ ] 필드/골대 보정을 통한 픽셀-미터 변환 알고리즘 작성
- [ ] 거리 및 프레임 변화량을 기반으로 한 공속도 추정 로직 구현
- [ ] 포즈 데이터(각도, 신체 밸런스 등) 기반 프리킥 스코어 산출 시스템 구현
- [ ] 분석 결과를 사용자가 이해하기 쉬운 텍스트(개선점 3가지 등)로 변환하는 모듈 구현

## Phase 4: 결과 화면 및 공유 기능 (1차 MVP)
- [ ] 분석 결과를 시각적으로 보여주는 리포트 대시보드 화면 개발
- [ ] 주요 스틸컷/타깃맵을 합성하여 공유용 이미지 카드(HTML to Canvas 등) 생성 기능 추가
- [ ] 소셜 공유 API(Web Share API) 연동

## Phase 5: 메타데이터 클라우드 저장 및 백엔드 (1차 MVP ~ 2차 MVP 전환)
- [ ] 익명/소셜 로그인(Auth) 연동
- [ ] 데이터베이스(Firebase/Supabase/Prisma) 연동 및 JSON 스키마 설계
- [ ] 사용자 동의를 기반으로 한 분석 결과 메타데이터 서버 전송 및 저장 로직 구현

## Phase 6: 2차/3차 MVP 확장 (Future)
- [ ] 사용자별 누적 기록 비교, 거리별 랭킹 시스템 구축
- [ ] 수동 공 좌표 보정 UI 개발
- [ ] 코치 모드 및 고급 분석/궤적 시뮬레이션 개발
